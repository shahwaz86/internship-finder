## Project Description

This project is a REST API that helps students find internships based on their skills.

The API allows users to:

- Add internship opportunities
- Search internships by skill
- Get internship recommendations
- Delete internships

## Technologies

- Node.js
- Express.js
- REST API

## API Endpoints

GET /api/internships → Get all internships

POST /api/internships → Add internship

GET /api/internships/search?skill=node → Search by skill

POST /api/recommend → Get internship recommendations

DELETE /api/internships/:id → Delete internship
